# SpritePacker

A client-side texture atlas generator for Game Developers.

## Features
- Pack sprites into optimal texture atlases (MaxRects algorithm).
- Run entirely in the browser (Web Worker driven).
- Export to Unity, Unreal Engine, and Godot.
- Dark mode UI built with Tailwind CSS.

## Tech Stack
- React 19 + TypeScript + Vite
- Zustand (State Management)
- maxrects-packer
- JSZip
- Tailwind CSS

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
